Ma Page HTML Pour Tous

Ma page html vous permet de mettre en pratique les concepts fondamentaux d html a travers une serie d exercice .

L objectif de la carte mentale est de permettre aux étudients de créer une carte mentale detaillée qui récapitule les points clés.

HTML acronyme de HypertText Markup Language "Langage de Balisage Hypertexte"est la base de tout site web.C est une forme de langage informatique mais il ne s agit pas d un langage de programmation.

HTML est un langage de balisage standard qui définit la structure et le contenu des pages web en utilisant des balises qui pécise le type de l élement integrer dans la page et comment il doit etre reconu par le navigateure.

Bien que le HTML soit la base de la structure et du contenu,il peut etre encore plus engonomique et interactif lorsqu il est combiné avec d autre langage tel que CSS=Cascading Style Sheets ,pour styliser les pages HTML en contolant les couleurs ,les polices ,les marges et d autres aspects visuels .
